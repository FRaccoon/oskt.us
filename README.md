# oskt.us

OSK team site.

## Developing

```
yarn
yarn run build
yarn start
```

Start a local server at `localhost:9966`.

Edit something.

## Deployment

Edit `circle.yml` and fill in the `GIT_****` fields.

Open [circleci](https://circleci.com) and build project.

Project Settings -> Checkout SSH Keys -> Add user key

Then, rebuild.
